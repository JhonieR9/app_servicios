4. Copia la URL que empieza con `https://`

## 🖥️ Configurar PC para 24/7

### Windows 10/11:

1. **Evitar suspensión:**
   - Presiona `Windows + I` (Configuración)
   - Ve a: Sistema → Energía y suspensión
   - Suspender el equipo: **Nunca**
   - Apagar pantalla: **30 minutos** (opcional)

2. **Evitar actualizaciones automáticas:**
   - Configuración → Windows Update
   - Pausar actualizaciones por 7 días

3. **Mantener conexión estable:**
   - Usa cable ethernet (mejor que WiFi)
   - Verifica que el router esté estable

## ⚠️ Solución de Problemas

### Si el servidor se detiene:

1. Cierra ambas ventanas (servidor y ngrok)
2. Vuelve a abrir `mantener_servidor.bat`
3. Vuelve a abrir `mantener_ngrok.bat`
4. Copia la nueva URL de ngrok
5. Actualiza los enlaces compartidos

### Si no puedes acceder desde internet:

1. Verifica que ambas ventanas estén abiertas
2. Verifica tu conexión a internet
3. Prueba la URL en modo incógnito
4. Verifica que ngrok esté autenticado

### Si hay muchos registros falsos:

1. Ve al panel de admin
2. Elimina los registros falsos
3. Considera agregar CAPTCHA (te puedo ayudar)
